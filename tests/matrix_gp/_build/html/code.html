Auto Generated GP Test Documentation¶

Run the test by going into the matrix_gp directory, and run:

nosetests -c gp.cfg –with-html –html-file results.html

Smoke Tests¶

Smoke tests for GP covariande functions¶

The test functions do not take any input. We fix both, data vectors and hyper-parameters. The expected results were computed by hand and with MATLAB.

We are using \(\mathbf{x} = [1.3, -2, 0]^T\) and \(\mathbf{y} = [1.3, -3.2]^T\) to test both \(\mathbf{K}(\mathbf{x},\mathbf{x})\) and \(\mathbf{K}(\mathbf{x},\mathbf{y})\). The hyper-parameters are defined as follows:

\(\sigma = 2.1\)

\(\ell= 1.8\)

\(p = 0.9\)

\(\alpha= 0.8\)

\(\sigma = 2.1\)

test_matrix_covs.test_linear(*args)¶

Tests the linear covariance LIN \(=\sigma^2 (x x^\prime)\)

test_matrix_covs.test_noise(*args)¶

Tests the noise covariance WN \(=\sigma^2 \delta_{x,x^\prime}\)

test_matrix_covs.test_periodic(*args)¶

Tests the periodic covariance PER \(= \sigma^2 \exp \bigg( \frac{2 \sin^2 ( \pi (x - x^\prime)/p}{\ell^2} \bigg)\)

test_matrix_covs.test_rationalQuadratic(*args)¶

Tests the rational quadratic covariance RQ \(=\sigma^2 \bigg(1 + \frac{(x - x^\prime)^2}{2 \alpha \ell^2} \bigg)^{-\alpha}\)

test_matrix_covs.test_squaredExponential(*args)¶

Tests the squared-exponential covariance SE \(= \sigma^2 \exp{-\frac{(x-x^\prime)^2}{2\ell^2}}\)

Tests for GP Inference¶

We do both, smoke tests and statistical tests¶

We start with testing the parameters of the multivariate normal, as determined by the GP and some observations.

test_matrix_gp.testNormalParameters(*args)¶

Training data: \(\mathbf{x} =[1.3, {-2}, 0]^T\)

\(\mathbf{y} =[5, 2.3, 8.2]^T\)

Unseen data:

\(\mathbf{x}^* =[1.4, {-3.2} ]^T\)

We provide a squared-exponential covariance matrix with \(\sigma = 2.1\) and \(\ell=1.8\).

test_matrix_gp.testOneSample(*args)¶

Training data:

\(\mathbf{x} =[1.3, {-2}, 0]^T\)

\(\mathbf{y} =[5, 2.3, 8.2]^T\)

Unseen data:

\(x^* =1.4\)

We provide a squared-exponential covariance matrix with \(\sigma = 2.1\) and \(\ell=1.8\).

We use a one-sample t-test to test if the samples are not different from a Gaussian with the mean that was analytically computed by hand. We also add some sanity checks in the end. Future applications of this smoke test may reveal it to be too conservative.

test_matrix_gp.testTwoSamples(*args)¶

Training data:

\(\mathbf{x} =[1.3, {-2}, 0]^T\)

\(\mathbf{y} =[5, 2.3, 8]^T\)

Unseen data with low covariance:

\(\mathbf{x}^* =[1.4,-20]\)

Unseen data with high covariance:

\(\mathbf{x}^* =[1.4,1.5]\)

We provide a squared-exponential covariance matrix with \(\sigma = 2.1\) and \(\ell=1.8\). We compute Pearson’s r and test for significance to check whether covariance is high with two datapoints where we expect it and where we do not expect.

test_matrix_gp.test_gp_inference_known_example(*args)¶
Inference Quality Smoke Test

We take a simple data generating process with a clear structure:

\(f(x)= 2x + 5 \sin 3x + \eta\) with \(\eta \sim \mathcal{N}(0,1)\).

We learn the hyper-parameters for K = LIN + PER + WN and extrapolate. We perform ad-hoc tests (as statistical tests for an extrapolation seem to be too conservative). We use 100 data points for training data.
